# Python图书管理系统 python257

## 项目概述

本项目是一个基于Python语言的图书管理系统，采用Flask作为Web开发框架，搭配MySQL数据库进行数据存储，使用HTML、CSS和JavaScript进行前端设计。

## 技术栈

- 后端：Python + Flask
- 数据库：MySQL
- 前端：HTML + CSS + JavaScript

## 功能模块

### 基础功能

- 登录注册
- 用户权限管理

### 图书管理

- 图书信息添加
- 图书信息修改
- 图书信息查询
- 图书信息删除

### 借阅管理

- 图书借阅记录

### 读者信息

- 读者信息查询

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 操作系统：兼容Windows、Linux、MacOS
- Python版本：3.x
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 安装Python环境
2. 安装Flask框架
3. 安装MySQL数据库
4. 配置数据库连接
5. 运行Web应用
